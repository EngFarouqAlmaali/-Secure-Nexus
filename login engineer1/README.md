## ✨ Overview

Secure Nexus is an elegant, responsive authentication system featuring a glass-morphism design, smooth animations, and comprehensive security measures. It provides seamless login/signup functionality with social media integration and client-side protection against common web vulnerabilities.

## 🚀 Features

- **🔄 Dual Authentication Modes**: Toggle between Login and Signup forms with fluid animations
- **🌐 Social Login Integration**: Support for Google, Facebook, LinkedIn, GitHub, and Microsoft OAuth
- **🛡️ Security First**: Built-in protection against XSS, CSRF, and SQL Injection attempts
- **🎨 Modern UI/UX**: Glass-morphism design with GSAP-powered animations
- **📱 Responsive Design**: Optimized for desktop and mobile devices
- **👁️ Password Visibility Toggle**: Interactive eye icons for password fields
- **⚡ Real-time Validation**: Instant feedback on form inputs
- **🎭 Dynamic Code Display**: Animated code snippet showcase on the interface

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Animations**: GSAP (GreenSock Animation Platform)
- **Icons**: Remix Icons
- **Fonts**: Google Fonts (Inter & Space Grotesk)
- **Security**: Custom client-side protection scripts

## 🚀 Getting Started

1. **Clone the repository**:

   ```bash
   git clone https://github.com/EngFarouqAlmaali/-Secure-Nexus.git
   cd secure-nexus
   ```

2. **Open in browser**:
   - Simply open `index.html` in your preferred web browser
   - No build process or dependencies required!

3. **Customize**:
   - Modify `assets/css/styles.css` for styling changes
   - Update `js/main.js` for interaction logic
   - Enhance security in `security/security.js`

## 🔒 Security Features

- **XSS Protection**: HTML escaping for user inputs
- **CSRF Simulation**: Token-based protection mechanism
- **SQL Injection Prevention**: Pattern detection and input sanitization
- **Input Validation**: Real-time form validation with error messaging

## 🎯 Usage

- **Login**: Enter email and password, click "Log In"
- **Signup**: Switch to signup mode, fill required fields, click "Sign Up"
- **Social Auth**: Click any social icon to simulate OAuth login
- **Password Toggle**: Click the eye icon to show/hide passwords
